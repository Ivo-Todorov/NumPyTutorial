This tutorial consists of the following parts:

 What is NumPy?
 How to create NumPy arrays
 Indexing, Fancy Indexing
 Slicing
 Universal Functions (Ufuncs)
 Broadcasting
 Masking, Sorting and Comparison
 Further learning
